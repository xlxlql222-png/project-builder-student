# 심쿵! 로맨스 주인공 테스트 - Blueprint ( 가독성 및 모바일 최적화)

## 개요
사용자 편의성과 애드센스 승인 요건을 모두 충족하도록 설계된 **수직형 단일 컬럼 로맨스 심리 테스트** 서비스입니다. PC와 모바일 환경 모두에서 끊김 없는 가독성을 제공합니다.

## 최적화 포인트 (v2.0)
- **수직형 단일 컬럼 레이아웃:** PC에서의 3단 분할 느낌을 제거하고, 모든 콘텐츠를 중앙으로 집중시켜 시선의 흐름을 수직으로 통일했습니다.
- **가독성 강화 (Contrast):** 배경색과 글자색의 간섭을 해결하기 위해 컨테이너의 불투명도를 높이고 텍스트 대비를 상향 조정했습니다.
- **정보성 가이드 섹션:** 시작 화면에 수직 흐름의 서비스 안내(Guide) 섹션을 추가하여 정보성 콘텐츠의 양을 보강했습니다.
- **성별 선택 UI 개선:** 나란히 배치되었던 성별 선택 버튼을 수직으로 쌓아 클릭 미스를 방지하고 텍스트 설명을 추가했습니다.
- **모바일 환경 최적화:** 터치하기 편한 버튼 크기와 스크롤에 최적화된 요소 배치를 적용했습니다.

## 기술 스택 및 서비스
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **UX/UI:** 1단 수직 레이아웃, 명도 대비 최적화, 파스텔 톤 큐트 테마
- **Services:** Google AdSense (수익화), Formspree (문의), Disqus (소통)

## 최근 변경 사항
- 3단 레이아웃을 1단 중앙 집중형 레이아웃으로 전면 개편.
- 텍스트 가독성 및 명도 대비 문제 해결.
- 시작 화면 안내 섹션 추가 및 수직 흐름 강화.
