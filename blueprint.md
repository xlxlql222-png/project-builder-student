# 로맨스 여주인공 테스트 - Blueprint (High-Quality Optimized)

## 개요
본 사이트는 구글 애드센스 승인 가이드를 준수하여 설계된 **고품질 심리 테스트 플랫폼**입니다. 10-20대 여성 사용자를 타겟으로 하며, 독창적인 콘텐츠와 최적의 사용자 경험(UX)을 제공합니다.

## 주요 기능 및 최적화 포인트
- **심리 테스트 (Core):** 정교하게 설계된 연애 성향 질문과 결과 알고리즘.
- **오늘의 행운 아이템 (Unique Point):** 매일 랜덤하게 변하는 로맨스 행운 아이템을 제공하여 재방문 가치를 높임.
- **정보성 콘텐츠:** 테스트 안내 및 과학적 배경 설명을 추가하여 구글이 선호하는 텍스트 풍성도를 확보.
- **신뢰성 (Transparency):** 푸터에 이용약관, 개인정보처리방침, 운영자 문의(Contact) 링크를 배치.
- **고급 디자인:** 반응형 레이아웃, 세련된 타이포그래피, 마이크로 인터랙션 적용.
- **수익화:** Google AdSense 최적화 (광고 스크립트 및 ads.txt 설정 완료).

## 기술 스택
- **Frontend:** HTML5 (Semantic), CSS3 (Modern Grid/Flex), Vanilla JS
- **Fonts:** Noto Sans KR (가독성 최적화), Poppins
- **Services:** Formspree, Disqus, Google AdSense

## 최근 변경 사항
- 애드센스 승인을 위한 고품질 사이트 최적화 (푸터, 정보 섹션 추가).
- 독창적 포인트 '오늘의 로맨스 행운 아이템' 기능 구현.
- 접근성 및 가독성 개선 (Progress Text, Badge 등 UI 요소 추가).
