# 심쿵! 로맨스 주인공 테스트 - Blueprint (지능형 하이브리드 최적화)

## 개요
모바일과 PC 환경에서 각각 최적화된 사용자 경험을 제공하기 위해 **지능형 하이브리드 뷰 시스템**이 적용된 로맨틱 심리 테스트 서비스입니다.

## 최적화 포인트 (v3.0)
- **지능형 자동 전환:** 접속 시 기기 타입(모바일/태블릿/PC) 및 해상도를 자동 감지하여 최적의 모드로 설정합니다.
- **모바일 전용 모드 (App-Like Mode):** 모바일 기기 접속 시 배경 블롭을 제거하고 풀스크린 레이아웃을 적용하여 앱과 같은 몰입감을 제공합니다.
- **수동 뷰 모드 토글:** 사용자가 원할 경우 상단 버튼을 통해 'PC 모드'와 '모바일 모드'를 실시간으로 전환할 수 있습니다.
- **가독성 및 대비(Contrast) 극대화:** 텍스트 색상을 어두운 계열(#1a1a1a)로 상향 조정하고 배경과의 간섭을 최소화하여 어떤 기기에서도 선명한 가독성을 보장합니다.
- **수직형 중앙 레이아웃:** 시선이 분산되지 않도록 1단 중앙 집중형 구조를 유지하며, 모바일 터치 영역을 확대했습니다.

## 기술 스택
- **Frontend:** HTML5, CSS3, JS (Vanilla)
- **Features:** Auto-Device Detection, Mode Toggle System, High Contrast Design
- **Services:** AdSense, Formspree, Disqus, Google Analytics (G-DJTQJCZ2KJ), MS Clarity (vmnqiyof3r)

## 최근 변경 사항
- MS Clarity 연동을 통한 사용자 유입 및 행동 히트맵 분석 도구 추가.
- Google Analytics (gtag.js) 연동으로 사용자 행동 데이터 수집 기반 마련.
- 기기 자동 감지 및 PC/Mobile 모드 전환 시스템 구축.
- 모바일 전용 앱 스타일 레이아웃 구현.
- 텍스트 명도 대비 및 가독성 문제 완전 해결.
